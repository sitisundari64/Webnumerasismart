📖 Konsep Penting

Pahami tiga ukuran pemusatan data berikut untuk menganalisis data nyata.

📐

RATA-RATA

Rata-rata

Jumlah semua data ÷ banyaknya data. Cocok untuk data terdistribusi normal.

Rata-rata = Σx / n

📏

MEDIAN

Nilai Tengah

Nilai tengah setelah data diurutkan. Lebih representatif bila ada outlier.

n ganjil: x[(n+1)/2]

📊

MODUS

Nilai Terbanyak

Nilai yang paling sering muncul. Berguna untuk data kategoris.

Mo = f(max)
